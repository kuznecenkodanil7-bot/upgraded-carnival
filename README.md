# Astra Visuals

Клиентский визуальный Fabric-мод для **Minecraft Java Edition 1.21.11**. Он вдохновлён идеей аккуратного визуального клиента, но не содержит wallhack, скрытого ESP, автоматизации боя или обхода античита.

## Что уже есть

- Watermark с анимированным акцентом
- Инфопанель: FPS, ping, координаты, скорость
- Keystrokes: WASD, прыжок, shift, ЛКМ и ПКМ
- Armor HUD с предметами и прочностью
- Список активных эффектов и таймеры
- Компас и направление взгляда
- Target HUD только для сущности, находящейся под обычным прицелом
- Индикатор попадания прицела
- Виньетка при низком здоровье
- Четыре темы: Neon, Aurora, Crimson, Mono
- Настраиваемая прозрачность панелей
- Экран настроек по **Right Shift**
- Автоматическое сохранение настроек в `config/astravisuals.json`
- Русская и английская локализация

## Сборка

Понадобятся:

- JDK 21
- Gradle 9.4.1 либо IntelliJ IDEA с поддержкой Gradle
- Интернет при первой сборке, чтобы Gradle получил Minecraft, Yarn, Fabric Loader и Fabric API

В папке проекта:

```bash
gradle wrapper
./gradlew build
```

На Windows:

```bat
gradle wrapper
gradlew.bat build
```

Готовый файл появится в `build/libs/`.

## Запуск для разработки

```bash
./gradlew runClient
```

## Установка

1. Установите Fabric Loader для Minecraft 1.21.11.
2. Положите Fabric API и собранный `astravisuals-1.0.0.jar` в папку `mods`.
3. Запустите игру.
4. Откройте настройки клавиш и при необходимости переназначьте клавишу меню.
5. По умолчанию меню открывается на **Right Shift**.

## Безопасные границы

Target HUD использует только стандартную сущность под прицелом клиента. Мод намеренно не ищет игроков за стенами, не раскрывает скрытые сущности и не вмешивается в бой. Красивый HUD полезнее бана, хотя человечество периодически спорит и с этим.

## Структура

- `AstraVisualsClient` — инициализация, клавиша и HUD callback
- `HudRenderer` — все HUD-модули
- `VisualConfigScreen` — экран настроек
- `VisualConfig` / `ConfigManager` — JSON-конфиг
- `Theme` / `ColorUtil` — темы и цвета
